Markdown 学习

发表于 2018-03-25 | 分类于 软件操作 |

Markdown 是一种轻量级标记语言，它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的 XHTML（或者 HTML） 文档”。本篇文章主要参考的是 GitHub 风格的语法 GitHub Flavored Markdown，以下是自己整理的一些内容，方便查阅。
